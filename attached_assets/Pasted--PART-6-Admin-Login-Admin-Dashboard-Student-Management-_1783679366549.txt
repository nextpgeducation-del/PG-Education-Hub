## Admin Authentication

Create a completely secure Admin Login System.

Only administrators should be able to access the Admin Dashboard.

Use Firebase Authentication, Firestore Database, secure sessions and role-based access control.

# First Login Security

When the Admin logs in for the first time:

* Force the Admin to change the default password.
* The default password must become invalid after it has been changed.
* Passwords must always be stored securely using hashing/authentication best practices.

# Security Requirements

* Only Admin can access the Admin Dashboard.
* Students must never access Admin pages.
* Protect all admin routes using authentication and role-based authorization.
* Use Firestore security rules.
* Log important admin actions for auditing (student updates, terminations, fee changes, uploads).

The Admin Dashboard must be responsive, fast, modern, production-ready and integrated with Firebase Firestore for real-time data updates.
