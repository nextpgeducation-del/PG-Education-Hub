# PART – 5 : Student Registration, Student Login & Student Dashboard

## Student Registration System

Create a premium, secure and fully responsive Student Registration page.

The registration form must use Firebase Authentication and Firestore Database.

---

# Registration Form Fields

## Required Fields

* Student Full Name
* Mobile Number
* Father's Name
* PIN Code
* School Name
* Class (Dropdown)
* Password
* Confirm Password

## Optional Field

* Email Address (Optional)

If the student does not enter an email address, registration must still be successful.

---

# Class Dropdown

Create the following options:

* Class 6–8
* Class 9–10
* Class 11–12

---

# Automatic Data Generation

After successful registration automatically generate:

* Student ID
* Admission Number
* Registration Date
* Registration Time
* Admission Date
* Account Status = Active
* Fee Dashboard (12 Months)
* Student Profile

Save everything inside Firebase Firestore.

---

# Registration Validation

Validate:

Student Name

Required

Mobile Number

Exactly 10 digits

PIN Code

Exactly 6 digits

Password

Minimum 8 Characters

Password Rules

* Uppercase Letter
* Lowercase Letter
* Number
* Special Character

Confirm Password

Must Match Password

Email

Validate only if entered.

---

# Registration Success

After successful registration:

Display

Registration Successful

Then automatically:

Save student information.

Create Student Dashboard.

Create Fee Dashboard.

Create Student ID.

Add student into Registered Students inside Admin Dashboard immediately.

No manual approval required.

# Security

Students must only access their own dashboard.

No student should be able to view another student's information.

All routes should be protected using Firebase Authentication and Firestore security rules.

The dashboard should be fully responsive, modern, fast-loading and production-ready with reusable components and Bootstrap 5.
