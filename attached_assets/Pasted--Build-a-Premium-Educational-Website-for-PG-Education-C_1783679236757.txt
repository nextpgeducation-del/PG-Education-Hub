# Build a Premium Educational Website for "PG Education"

Create a **modern, premium, fully responsive Educational Coaching Management Website** named **PG Education**.

The website should look like a professional coaching institute with a clean UI, premium design, smooth animations, fast loading speed, accessible components, and production-ready architecture.

The project must be built using modern web development standards and should be suitable for deployment.

* Contact coaching

Student can:

* Register

* Access Student Dashboard
* Download Notes

Admin can:

* Login
